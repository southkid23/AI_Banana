Windows:

1. Launch command prompt on the project folder "AI_Banana" or navigate 
to the project folder using the command prompt

2. to run the program:
   Type: "go <folder name>" e.g. "go phase_4"

//////////////////////////////////////////////////////////////////////
//////////////////////////////////////////////////////////////////////
//////////////////////////////////////////////////////////////////////

Mac OSX:

1. Launch terminal on the project folder "AI_Banana" or navigate to 
the project folder using terminal
	e.g. "cd Desktop/AI_Banana"

2. to run the program:
   type: "./go_mac"

3. On the terminal, there will be a prompt for which phase to run
	- Type '2' to run phase 2
